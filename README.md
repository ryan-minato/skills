# skills

A collection of skills focused on AI training, software dev, and data analytics.

## Repository Structure

| Path | Purpose |
|---|---|
| `skills/` | Installable skills; one directory per skill |
| `docs/specs/` | Per-skill intent definitions |
| `docs/` | Knowledge base: quality standards, workflow, references |
| `scripts/` | Repo-level validators and formatters |
| `.agents/skills/` | Symlink → `skills/`; for skill self-application |

## Contributing

- Agents: start from [AGENTS.md](AGENTS.md).
- Authors: read [docs/WORKFLOW.md](docs/WORKFLOW.md) for the authoring process and
  [docs/QUALITY.md](docs/QUALITY.md) for acceptance criteria.
