# Gitmoji — Full Official List

Source: https://gitmoji.dev/ — every official gitmoji, grouped by intent. When two
candidates remain after the SKILL.md selection list, prefer the more specific one.

## Features, fixes, and behavior

| Emoji | Code | When to use |
|---|---|---|
| ✨ | `:sparkles:` | Introduce new features |
| 🐛 | `:bug:` | Fix a bug |
| 🚑️ | `:ambulance:` | Critical hotfix |
| 🩹 | `:adhesive_bandage:` | Simple fix for a non-critical issue |
| 🥅 | `:goal_net:` | Catch errors |
| 💥 | `:boom:` | Introduce breaking changes |
| 🚩 | `:triangular_flag_on_post:` | Add, update, or remove feature flags |
| 👽️ | `:alien:` | Update code due to external API changes |
| 👔 | `:necktie:` | Add or update business logic |
| 🦺 | `:safety_vest:` | Add or update code related to validation |
| ⏪️ | `:rewind:` | Revert changes |
